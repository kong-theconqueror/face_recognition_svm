# face_recognition_svm
Facial recognition and authentication system based on face embedding and SVM


## Install environments
### 1. install CMake
Download CMake:
https://cmake.org/download/

Choose:
Windows x64 Installer (.msi)

While installing:
Select Add CMake to system PATH

### 2. install Visual Studio Build Tools
Install Visual Studio Build Tools

Download:
https://visualstudio.microsoft.com/visual-cpp-build-tools/

Choose:
- Desktop development with C++
- MSVC v143
- Windows SDK

### 3. Install pyhton virtualenv
```
python -m pip install virtualenv

python -m virtualenv .venv

.venv\Scripts\active

pip install -r requirements.txt
```

## Run labs
### 1. Set Jupyter Notebook password
```
jupyter notebook password
```

### 2. Start Jupyter Notebook
```
jupyter notebook
```
